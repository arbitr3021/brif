3 Мета та завдання гри
Оберіть одну або кілька цілей, які має виконувати гра:
  • Збір контактних даних потенційних клієнтів (лідогенерація)
  • Підвищення впізнаваності бренду та продукту
  • Освітня або інформаційна функція
  • Просування конкретного продукту або акційної пропозиції
  • Залучення та утримання аудиторії
  • Інше:
Що гравець має запам'ятати після завершення гри?

5 Ігрова механіка
Тип ігрової механіки
Оберіть бажаний тип або опишіть власний варіант:
  • Клікер — тапай або клікай по об'єктах, щоб знищити загрози
  • Збирання — ловити падаючі предмети у кошик або контейнер
  • Вікторина — запитання з варіантами відповідей
  • Стратегія — розподіл ресурсів по ігровому полю (покрокова)
  • Пазл — знайди пару, склади зображення
